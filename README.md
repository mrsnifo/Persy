# Presy

Manage all your people in one place.

![Presy Preview](https://github.com/user-attachments/assets/b3b60ab0-ec50-45dd-803d-e65625e63dd9)

## Technologies

**Frontend:** Preact, Deno Fresh, Tailwind CSS, DaisyUI\
**Backend:** Java, Jersey, Hibernate, Grizzly

## Configuration

### Backend Environment Variables

- DB_URL
- DB_USER
- DB_PASSWORD
- SERVER_PORT
- ALLOWED_ORIGINS

### Frontend Environment Variables

- VITE_API_BASE_URL
